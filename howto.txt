Schritt-für-Schritt-Anleitung: Dein n8n FTPS-Modul erstellen und verwenden

Diese Anleitung führt dich durch den Prozess, das FTPS-Modul in eine n8n-Entwicklungsumgebung zu integrieren.

**Voraussetzung:** Du hast Node.js (inkl. npm) und git auf deinem System installiert.

---
### Schritt 1: n8n-Entwicklungsumgebung einrichten
---

1.  **Öffne ein neues Terminal oder eine neue Kommandozeile.**

2.  **Klone das n8n-Repository von GitHub.** Dies lädt den gesamten Quellcode von n8n auf deinen Computer.
    ```bash
    git clone https://github.com/n8n-io/n8n.git
    ```

3.  **Wechsle in das neu erstellte n8n-Verzeichnis.**
    ```bash
    cd n8n
    ```

4.  **Installiere alle notwendigen Abhängigkeiten.** Dies kann einige Minuten dauern.
    ```bash
    npm install
    ```

5.  **Erstelle das Projekt zum ersten Mal.** Dieser Schritt kompiliert den Code und stellt sicher, dass alles korrekt eingerichtet ist.
    ```bash
    npm run build
    ```

---
### Schritt 2: Dein neues FTPS-Modul integrieren
---

1.  **Erstelle ein Verzeichnis für dein neues Modul.** Der Name des Verzeichnisses wird zum Namen des Moduls. Wir nennen es hier `Ftps`.
    *   Erstelle den folgenden Ordnerpfad innerhalb des `n8n`-Verzeichnisses: `packages/nodes-base/nodes/Ftps`

2.  **Verschiebe die von mir erstellten Dateien.**
    *   Verschiebe `Ftps.node.ts` nach `n8n/packages/nodes-base/nodes/Ftps/Ftps.node.ts`
    *   Verschiebe `FtpsApi.credentials.ts` nach `n8n/packages/nodes-base/credentials/FtpsApi.credentials.ts`

3.  **Füge die `basic-ftp`-Abhängigkeit hinzu.** Dein Modul benötigt diese Bibliothek, um mit FTPS-Servern zu kommunizieren.
    *   Öffne die Datei `n8n/packages/nodes-base/package.json`.
    *   Finde den Abschnitt `"dependencies"`.
    *   Füge die folgende Zeile hinzu (achte darauf, das Komma in der Zeile davor zu setzen, falls nötig):
        ```json
        "basic-ftp": "^5.0.0"
        ```
    *   Speichere die Datei.

---
### Schritt 3: n8n neu erstellen und starten
---

1.  **Installiere die neue Abhängigkeit.** Führe im Hauptverzeichnis von `n8n` den folgenden Befehl aus, um `basic-ftp` herunterzuladen:
    ```bash
    npm install
    ```

2.  **Erstelle das n8n-Projekt erneut.** Dieser Befehl kompiliert dein neues Modul zusammen mit dem Rest von n8n.
    ```bash
    npm run build
    ```

3.  **Starte n8n im Entwicklermodus.**
    ```bash
    npm run start
    ```

---
### Schritt 4: Dein FTPS-Modul in n8n verwenden
---

1.  **Öffne n8n in deinem Browser.** Normalerweise ist es unter `http://localhost:5678` erreichbar.

2.  **Erstelle einen neuen Workflow.**

3.  **Füge deine FTPS-Anmeldeinformationen hinzu.**
    *   Gehe zu "Credentials" in der linken Seitenleiste.
    *   Klicke auf "Add credential".
    *   Suche nach "FTPS API" und wähle es aus.
    *   Fülle die Felder (Host, User, Password etc.) aus und speichere.

4.  **Füge deinen FTPS-Knoten zum Workflow hinzu.**
    *   Klicke auf das `+`-Symbol im Workflow-Editor.
    *   Suche nach dem "FTPS"-Knoten und füge ihn hinzu.
    *   Wähle unter "Credentials" deine soeben erstellten Anmeldeinformationen aus.
    *   Konfiguriere die Felder "Local File Path" und "Remote File Path".
    *   Führe den Knoten aus, um eine Datei hochzuladen.

Das war's! Du hast erfolgreich ein benutzerdefiniertes n8n-Modul erstellt und in Betrieb genommen.
